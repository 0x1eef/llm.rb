## About

The following is a minimal streaming chat application built with
[llm.rb](https://github.com/llmrb/llm.rb), Rack, and WebSockets.

## Stack

- rack
- llm.rb
- falcon
- async-websocket
- plain HTML, CSS, and JavaScript
